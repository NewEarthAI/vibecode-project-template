# Found friction? Tell us — it becomes a fix

This template gets better through exactly one loop: **you hit friction → we hear
about it → the fix ships to everyone via `/update-latest`.** There is no forum,
no community portal, no marketplace — on purpose. Just a short path from your
annoyance to a shipped improvement.

## The 30-second way

Open a GitHub issue on the template repo with three lines:

1. **What you were trying to do** (one sentence)
2. **What happened instead** (paste the error or describe the confusion)
3. **How annoying it was** (mildly / quite / rage-quit)

No formatting requirements. Plain English is perfect.

## The zero-effort way

If you got this template from someone you know, just message them the same
three lines. They'll route it in.

## Contributing an actual fix

If you've already fixed or improved something in your copy, run
**`/push-to-template`** — it strips out everything project-specific and
prepares the improvement as a contribution. Ordinary pull requests are the
whole contribution model for now; a skills marketplace is deliberately
deferred until there are enough adopters to justify one.
